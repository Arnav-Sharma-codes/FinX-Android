# FinX 💹
### AI-Powered Financial Intelligence Platform for Android

FinX is a modern Android application that combines real-time market data with multiple AI models to help investors understand markets faster through intelligent insights, news analysis, portfolio intelligence, and personalized financial recommendations.

> Built using Kotlin + Jetpack Compose + Multiple AI APIs.
