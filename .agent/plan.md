## Project Brief

# FinX Project Brief

FinX is a modern fintech application designed to simplify financial planning. The app
 bridges the gap between complex market data and actionable insights, providing users with a clear, energetic interface to manage their investment strategies.

## Features
*   **Live Market Pulse**: A real-time dashboard fetching the latest market data and stock performance, presented
 through vibrant Material 3 components.
*   **Adaptive Comparison Engine**: A tool to compare stocks and markets side-
by-side, leveraging adaptive layouts to ensure a seamless experience across phones, foldables, and tablets.
*   **Plain-Language Insights**: An AI-driven feature that translates complex financial metrics and market jargon into simplified, easy-to
-understand summaries.
*   **Smart Investment Advisor**: Tailored investment recommendations generated by analyzing current market trends and historical
 data.

## High-Level Tech Stack
*   **Language**: Kotlin
*   **UI Framework**: Jetpack
 Compose with **Material Design 3 (M3)**
*   **Navigation**: **Jetpack Navigation 3**
 (State-driven)
*   **Adaptive Layouts**: **Compose Material Adaptive** library for multi-pane and
 responsive design.
*   **Concurrency**: Kotlin Coroutines & Flow for reactive data streams.
*   **Networking**: Retrofit
 & OkHttp for market data API integration.
*   **Serialization**: Moshi for JSON parsing.
